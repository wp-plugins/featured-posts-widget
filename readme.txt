A Wordpress plugin to create a Featured Posts widget

== Description ==

A Wordpress plugin to create a Featured Posts widget

Features:
* Customisable title
* Select categories to choose posts from
* Customisable thumbnails (none, above, below, left, right)
* Customisable thumbnail sizes
